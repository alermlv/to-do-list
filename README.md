# To-Do List

Simple task manager built with vanilla JavaScript.  
Tasks can be marked as important, scheduled by date, completed, and stored locally in the browser.

## Features

- Create tasks with name, description, and date
- Mark tasks as important
- Mark tasks as completed
- Automatic task grouping (Important, Today, Tomorrow, Date, Completed)
- Task sorting by priority and date
- LocalStorage persistence
- Accessible controls with ARIA attributes

## Tech

- HTML
- CSS
- Vanilla JavaScript
